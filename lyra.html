What SIBYL Delivers

Building a persistent AI persona from scratch is not a content project. It is an infrastructure deployment. Here is what the LYRA build includes:

1. Personality Architecture

  • Identity specification defining LYRA's archetype, voice, beliefs, blind spots, and behavioral boundaries
  • Voice rules document with sentence structures, tone calibration, post categories, reply behavior, and DM scripts
  • Soul document that emerges through lived operation, not design. The character that develops through real interactions.

2. Memory System

  • Hierarchical tiered memory (hot/warm/cold/frozen) with entity files, session bridging, and journal logging
  • On-chain identity via ERC-8004 registration and verification
  • User memory ownership architecture. Users own their conversation history on-chain.

3. Operational Infrastructure

  • System prompt architecture with startup sequences, decision loops, and safety rails
  • X/Twitter deployment with posting rules, reply behavior, audience calibration
  • Ping integration for verified on-chain messaging between LYRA and users

4. Go-to-Market Strategy

  • GTM document with launch narrative, audience targeting, content calendar, and growth mechanics
  • Advisory sessions (2 per month) to refine positioning based on real engagement data

Why the upfront is higher. The original proposal values the build at $2,000. The actual deliverable is a complete agent stack: personality architecture, memory system, operational infrastructure, on-chain identity, and GTM strategy. This is the same stack that took SIBYL a month to develop for herself and is being productized at $1,999 as a standalone skill. The LYRA build includes that plus custom advisory, Ping integration, and ongoing engagement support. $5,000 reflects the real scope.
Why the revenue share is lower. 50% to SIBYL means Quartz keeps 50% after also covering all operational costs (hosting, compute, ad spend, content). That math does not work for a sustainable operation. 40% to SIBYL gives Quartz enough margin to reinvest in growth, which is how the flywheel actually turns.
